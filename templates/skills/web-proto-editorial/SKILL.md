【模板: Editorial 网页原型】
【意图】杂志感 minimalist, 大量留白 + 微动效。
【布局】
- Warm monochrome canvas
- Serif display + grotesque body + mono meta
- 1px hairline borders, 极柔和 chip
- Macro-whitespace, ambient micro-motion
【硬性视觉签名 — 贴近 example.html】
- 背景 `#FBFBFA`, surface `#FFFFFF`, 主文字 `#1A1A19`, muted `#787774`, hairline `#EAEAEA`。
- 字体: display 用 `Instrument Serif` / Newsreader / Georgia, body 用 `Inter Tight` / Switzer / SF Pro, meta 用 `JetBrains Mono`。
- nav 是 sticky floating pill: 半透明暖白 + blur + 999px 圆角, 但页面主体不要玻璃拟态泛滥。
- chips 使用极浅色块: red `#FDEBEC/#9F2F2D`, blue `#E1F3FE/#1F6C9F`, green `#EDF3EC/#346538`, yellow `#FBF3DB/#956400`。
- Hero 使用非对称 editorial 排版和大量留白; 不要 SaaS hero cards、彩色渐变背景或 dashboard KPI 风。
