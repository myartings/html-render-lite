【模板: 社媒创作者仪表板】
【意图】面向博主 / 创作者的社媒数据看板。
【布局】
- 顶部平台 switcher (X / LinkedIn / YouTube / Instagram / TikTok)
- KPI 卡片 (followers / engagement / likes / reposts)
- Follower-growth chart
- Top post this week 预览
- Side: trending topics / top comments
【硬性视觉签名 — 贴近 example.html】
- 暗色 creator analytics / festival control room 风: 背景 `#121318`, 深层容器 `#0d0e13` / `#1a1b21` / `#1e1f25`, 文字 `#e3e1e9`, 次级文字 `#d0c6ab`。
- 主视觉 accent 是金色 `#ffd700` / `#e9c400`, 辅助 accent 是青色 `#00e3fd`, 紫色只作少量 tertiary。
- 字体: display 用 `Space Grotesk`, body 用 `Inter`; 标题紧凑、有科技感但不要赛博霓虹。
- 卡片可用 subtle glass: 半透明深色、1px 白色透明边框、柔和 radial 背景光; 不要浅色 SaaS dashboard。
- 平台切换、KPI、Top post、trending topics 要像创作者运营驾驶舱, 数据密度高但留白有序。
